# 🚀 WP Content AI Landing Page

Uma landing page moderna, persuasiva e otimizada para conversão, desenvolvida para o plugin **WP Content AI**. Este projeto destaca o uso de Inteligência Artificial para automação de conteúdo e SEO no WordPress.

## ✨ Funcionalidades Principais

- **Design Premium & Moderno**: Estética tecnológica com paleta dark mode (Deep Blue e Vibrant Green).
- **Interface Intuitiva**: Seções detalhadas sobre automação, SEO de alta performance e enriquecimento de mídia.
- **Micro-animações**: Experiência de usuário fluida e interativa.
- **Responsividade Total**: Otimizada para todos os dispositivos (Mobile-first).
- **Seções Estratégicas**:
  - Hero persuasivo com CTA clara.
  - Prova de autoridade e benefícios da IA.
  - Showcase da interface do plugin.
  - Tabela de preços simplificada.
  - FAQ detalhado.

## 🛠️ Tecnologias Utilizadas

- **React 19**: Framework para construção da interface.
- **Vite**: Ferramenta de build ultra-rápida.
- **Vanilla CSS**: Estilização personalizada para máxima performance e controle estético.
- **Lucide React**: Biblioteca de ícones modernos.

## 🚀 Como Executar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/acelinosilva/wp-content-ai-landing.git
   ```

2. **Instale as dependências**:
   ```bash
   cd wp-content-ai-landing
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```
